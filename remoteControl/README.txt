This is remote control for remote object (created to remotely control robots from your pc).
Control scripts provided by TRIK Studio (more about it: http://www.trikset.com/).
Remote control was created with Microsoft Visual Studio Professional 2013 (Update 2), 
using C# programming language. 
This project provides Windows Forms Application with buttons to control remote object (robot). 
Methods (connect to object, send message to object)
works asyncronously (more about it: https://msdn.microsoft.com/en-us/library/hh191443.aspx).
