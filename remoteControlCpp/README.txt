This is remote control for remote object (created to remotely control robots from your pc).
Control scripts provided by TRIK Studio (more about it: http://www.trikset.com/).
Remote control was created with QT Creator, using C++ programming language. 
This project provides GUI with buttons to control remote object (robot). 
This project was created using https://github.com/trikset/trik-models/tree/master/desktop-gamepad-demo
source files.
